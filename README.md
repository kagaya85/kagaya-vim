# Kagaya's nvim config

neovim config with lua plug-in

# How to use

## Requirement

- neovim 0.8.0+

## Installation

```
git clone git@github.com:kagaya85/kagaya.vim.git ~/.config/nvim
```

then run:

```
nvim
```
